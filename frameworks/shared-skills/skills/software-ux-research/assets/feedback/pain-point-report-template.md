## Template Usage Notes

**How to use this template**:
1. Copy entire document
2. Replace all `[bracketed]` placeholders
3. Delete sections not applicable
4. Add rows as needed to tables
5. Link to design pattern resources

**Output feeds**:
- `software-ui-ux-design` skill for pattern selection
- Product roadmap for feature prioritization
- Engineering for bug triage

**Update frequency**:
- Weekly for high-volume products
- Bi-weekly for medium volume
- Monthly for low volume
