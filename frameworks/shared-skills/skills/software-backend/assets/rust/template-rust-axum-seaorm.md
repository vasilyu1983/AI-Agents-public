# When to Use

Use this template when building:
- Performance-critical systems (<1ms latency requirements)
- High-throughput APIs (millions of requests/second)
- Memory-constrained environments (embedded, edge computing)
- Services requiring zero-downtime and fault tolerance
- Systems processing sensitive data (cryptography, financial transactions)

**Rust Advantages:**
- **Memory safety without GC**: No null pointer exceptions, use-after-free, or data races (compiler-enforced)
- **Zero-cost abstractions**: High-level code compiles to performance equivalent to hand-written C
- **Fearless concurrency**: Ownership system prevents data races at compile time
- **Blazing speed**: Performance comparable to C/C++ (often faster than Go/Java/Node.js)
- **Small binary size**: Static linking with musl produces tiny containers (<10MB)
- **Rich type system**: Algebraic data types, pattern matching, trait-based generics

```rust
use axum::{
    error_handling::HandleErrorLayer,
    http::StatusCode,
    routing::{get, post},
    BoxError, Router,
};
use std::time::Duration;
use tower::{
    buffer::BufferLayer,
    limit::RateLimitLayer,
    timeout::TimeoutLayer,
    ServiceBuilder,
};
use tower_http::{
    compression::CompressionLayer,
    cors::{Any, CorsLayer},
    trace::TraceLayer,
};
use tracing_subscriber::{layer::SubscriberExt, util::SubscriberInitExt};

mod api;
mod domain;
mod entities;
mod infrastructure;
mod utils;

use infrastructure::{config::Config, database};

#[tokio::main]
async fn main() -> anyhow::Result<()> {
    // Initialize tracing
    tracing_subscriber::registry()
        .with(
            tracing_subscriber::EnvFilter::try_from_default_env()
                .unwrap_or_else(|_| "info".into()),
        )
        .with(tracing_subscriber::fmt::layer().json())
        .init();

    // Load configuration
    let config = Config::from_env()?;
    tracing::info!("Configuration loaded: {}", config.app.name);

    // Connect to database
    let db = database::connect(&config.database).await?;

    // Connect to Redis
    let redis_client = redis::Client::open(config.redis.url.clone())?;
    let redis_conn = redis_client.get_multiplexed_tokio_connection().await?;

    // Build application state
    let app_state = api::AppState {
        db,
        redis: redis_conn,
        config: config.clone(),
    };

    // Build middleware stack
    let middleware = ServiceBuilder::new()
        .layer(HandleErrorLayer::new(|err: BoxError| async move {
            tracing::error!("Unhandled error: {}", err);
            StatusCode::INTERNAL_SERVER_ERROR
        }))
        .layer(BufferLayer::new(1024))
        .layer(RateLimitLayer::new(
            config.rate_limit.requests,
            Duration::from_secs(config.rate_limit.window),
        ))
        .layer(TimeoutLayer::new(Duration::from_secs(30)))
        .layer(TraceLayer::new_for_http())
        .layer(CompressionLayer::new())
        .layer(
            CorsLayer::new()
                .allow_origin(Any)
                .allow_methods(Any)
                .allow_headers(Any)
                .max_age(Duration::from_secs(config.cors.max_age as u64)),
        );

    // Build application routes
    let app = Router::new()
        .route("/health", get(health_check))
        .merge(api::routes::create_routes(app_state))
        .layer(middleware);

    // Start server
    let addr = format!("{}:{}", config.app.host, config.app.port);
    tracing::info!("Server starting on {}", addr);

    let listener = tokio::net::TcpListener::bind(&addr).await?;
    axum::serve(listener, app.into_make_service())
        .with_graceful_shutdown(shutdown_signal())
        .await?;

    Ok(())
}

async fn health_check() -> &'static str {
    "OK"
}

async fn shutdown_signal() {
    let ctrl_c = async {
        tokio::signal::ctrl_c()
            .await
            .expect("failed to install Ctrl+C handler");
    };

    #[cfg(unix)]
    let terminate = async {
        tokio::signal::unix::signal(tokio::signal::unix::SignalKind::terminate())
            .expect("failed to install signal handler")
            .recv()
            .await;
    };

    #[cfg(not(unix))]
    let terminate = std::future::pending::<()>();

    tokio::select! {
        _ = ctrl_c => {},
        _ = terminate => {},
    }

    tracing::info!("Shutdown signal received, starting graceful shutdown");
}
```

## src/utils/auth.rs

```rust
use argon2::{
    password_hash::{rand_core::OsRng, PasswordHash, PasswordHasher, PasswordVerifier, SaltString},
    Argon2,
};
use chrono::{Duration, Utc};
use jsonwebtoken::{decode, encode, DecodingKey, EncodingKey, Header, Validation};
use serde::{Deserialize, Serialize};
use thiserror::Error;

#[derive(Debug, Error)]
pub enum AuthError {
    #[error("Invalid password")]
    InvalidPassword,
    #[error("Token generation failed")]
    TokenGenerationFailed,
    #[error("Token verification failed")]
    TokenVerificationFailed,
    #[error("Password hashing failed")]
    PasswordHashingFailed,
}

#[derive(Debug, Serialize, Deserialize)]
pub struct Claims {
    pub sub: i32,        // user_id
    pub email: String,
    pub role: String,
    pub exp: i64,        // expiration
    pub iat: i64,        // issued at
}

pub struct JWTManager {
    secret: String,
    expiration: i64,
}

impl JWTManager {
    pub fn new(secret: String, expiration: i64) -> Self {
        Self { secret, expiration }
    }

    pub fn generate(&self, user_id: i32, email: String, role: String) -> Result<String, AuthError> {
        let now = Utc::now();
        let exp = (now + Duration::seconds(self.expiration)).timestamp();

        let claims = Claims {
            sub: user_id,
            email,
            role,
            exp,
            iat: now.timestamp(),
        };

        encode(
            &Header::default(),
            &claims,
            &EncodingKey::from_secret(self.secret.as_ref()),
        )
        .map_err(|_| AuthError::TokenGenerationFailed)
    }

    pub fn verify(&self, token: &str) -> Result<Claims, AuthError> {
        decode::<Claims>(
            token,
            &DecodingKey::from_secret(self.secret.as_ref()),
            &Validation::default(),
        )
        .map(|data| data.claims)
        .map_err(|_| AuthError::TokenVerificationFailed)
    }
}

pub fn hash_password(password: &str) -> Result<String, AuthError> {
    let salt = SaltString::generate(&mut OsRng);
    let argon2 = Argon2::default();

    argon2
        .hash_password(password.as_bytes(), &salt)
        .map(|hash| hash.to_string())
        .map_err(|_| AuthError::PasswordHashingFailed)
}

pub fn verify_password(hash: &str, password: &str) -> Result<(), AuthError> {
    let parsed_hash = PasswordHash::new(hash).map_err(|_| AuthError::InvalidPassword)?;

    Argon2::default()
        .verify_password(password.as_bytes(), &parsed_hash)
        .map_err(|_| AuthError::InvalidPassword)
}
```

## src/api/middleware/auth.rs

```rust
use axum::{
    extract::{Request, State},
    http::StatusCode,
    middleware::Next,
    response::Response,
    Json,
};
use serde_json::json;

use crate::{api::AppState, utils::auth::Claims};

pub async fn auth_middleware(
    State(state): State<AppState>,
    mut req: Request,
    next: Next,
) -> Result<Response, (StatusCode, Json<serde_json::Value>)> {
    let auth_header = req
        .headers()
        .get("Authorization")
        .and_then(|h| h.to_str().ok());

    let token = match auth_header {
        Some(header) if header.starts_with("Bearer ") => &header[7..],
        _ => {
            return Err((
                StatusCode::UNAUTHORIZED,
                Json(json!({"error": "Missing or invalid authorization header"})),
            ))
        }
    };

    let jwt_manager = crate::utils::auth::JWTManager::new(
        state.config.jwt.secret.clone(),
        state.config.jwt.expiration,
    );

    let claims = jwt_manager.verify(token).map_err(|_| {
        (
            StatusCode::UNAUTHORIZED,
            Json(json!({"error": "Invalid or expired token"})),
        )
    })?;

    // Store claims in request extensions
    req.extensions_mut().insert(claims);

    Ok(next.run(req).await)
}

pub fn get_claims(req: &Request) -> Option<&Claims> {
    req.extensions().get::<Claims>()
}
```

## src/api/handlers/auth.rs

```rust
use axum::{extract::State, http::StatusCode, Json};
use sea_orm::{ActiveModelTrait, ColumnTrait, EntityTrait, QueryFilter, Set};
use serde::{Deserialize, Serialize};
use validator::Validate;

use crate::{
    api::AppState,
    entities::{prelude::*, user},
    utils::{auth, errors::AppError},
};

#[derive(Debug, Deserialize, Validate)]
pub struct RegisterRequest {
    #[validate(email)]
    pub email: String,
    #[validate(length(min = 8))]
    pub password: String,
    #[validate(length(min = 2, max = 50))]
    pub name: String,
}

#[derive(Debug, Deserialize, Validate)]
pub struct LoginRequest {
    #[validate(email)]
    pub email: String,
    pub password: String,
}

#[derive(Debug, Serialize)]
pub struct AuthResponse {
    pub user: user::Model,
    pub token: String,
}

pub async fn register(
    State(state): State<AppState>,
    Json(req): Json<RegisterRequest>,
) -> Result<(StatusCode, Json<AuthResponse>), AppError> {
    // Validate request
    req.validate()?;

    // Check if user already exists
    let existing = User::find()
        .filter(user::Column::Email.eq(&req.email))
        .one(&state.db)
        .await?;

    if existing.is_some() {
        return Err(AppError::BadRequest("Email already in use".to_string()));
    }

    // Hash password
    let hashed_password = auth::hash_password(&req.password)?;

    // Create user
    let new_user = user::ActiveModel {
        email: Set(req.email.clone()),
        password: Set(hashed_password),
        name: Set(req.name),
        role: Set("user".to_string()),
        is_active: Set(true),
        ..Default::default()
    };

    let user = new_user.insert(&state.db).await?;

    // Generate JWT
    let jwt_manager = auth::JWTManager::new(
        state.config.jwt.secret.clone(),
        state.config.jwt.expiration,
    );
    let token = jwt_manager.generate(user.id, user.email.clone(), user.role.clone())?;

    Ok((
        StatusCode::CREATED,
        Json(AuthResponse { user, token }),
    ))
}

pub async fn login(
    State(state): State<AppState>,
    Json(req): Json<LoginRequest>,
) -> Result<Json<AuthResponse>, AppError> {
    // Validate request
    req.validate()?;

    // Find user
    let user = User::find()
        .filter(user::Column::Email.eq(&req.email))
        .one(&state.db)
        .await?
        .ok_or_else(|| AppError::Unauthorized("Invalid credentials".to_string()))?;

    // Verify password
    auth::verify_password(&user.password, &req.password)
        .map_err(|_| AppError::Unauthorized("Invalid credentials".to_string()))?;

    // Check if active
    if !user.is_active {
        return Err(AppError::Forbidden("Account is deactivated".to_string()));
    }

    // Generate JWT
    let jwt_manager = auth::JWTManager::new(
        state.config.jwt.secret.clone(),
        state.config.jwt.expiration,
    );
    let token = jwt_manager.generate(user.id, user.email.clone(), user.role.clone())?;

    Ok(Json(AuthResponse { user, token }))
}
```

# 8. Repository Pattern / Data Access

## src/domain/repositories/user.rs

```rust
use async_trait::async_trait;
use sea_orm::{DatabaseConnection, DbErr};

use crate::entities::{prelude::*, user};

#[async_trait]
pub trait UserRepository: Send + Sync {
    async fn create(&self, user: user::ActiveModel) -> Result<user::Model, DbErr>;
    async fn find_by_id(&self, id: i32) -> Result<Option<user::Model>, DbErr>;
    async fn find_by_email(&self, email: &str) -> Result<Option<user::Model>, DbErr>;
    async fn update(&self, user: user::ActiveModel) -> Result<user::Model, DbErr>;
    async fn delete(&self, id: i32) -> Result<(), DbErr>;
    async fn list(&self, offset: u64, limit: u64) -> Result<(Vec<user::Model>, u64), DbErr>;
}

pub struct UserRepositoryImpl {
    db: DatabaseConnection,
}

impl UserRepositoryImpl {
    pub fn new(db: DatabaseConnection) -> Self {
        Self { db }
    }
}

#[async_trait]
impl UserRepository for UserRepositoryImpl {
    async fn create(&self, user: user::ActiveModel) -> Result<user::Model, DbErr> {
        use sea_orm::ActiveModelTrait;
        user.insert(&self.db).await
    }

    async fn find_by_id(&self, id: i32) -> Result<Option<user::Model>, DbErr> {
        use sea_orm::EntityTrait;
        User::find_by_id(id).one(&self.db).await
    }

    async fn find_by_email(&self, email: &str) -> Result<Option<user::Model>, DbErr> {
        use sea_orm::{ColumnTrait, EntityTrait, QueryFilter};
        User::find()
            .filter(user::Column::Email.eq(email))
            .one(&self.db)
            .await
    }

    async fn update(&self, user: user::ActiveModel) -> Result<user::Model, DbErr> {
        use sea_orm::ActiveModelTrait;
        user.update(&self.db).await
    }

    async fn delete(&self, id: i32) -> Result<(), DbErr> {
        use sea_orm::{EntityTrait, ModelTrait};
        if let Some(user) = User::find_by_id(id).one(&self.db).await? {
            user.delete(&self.db).await?;
        }
        Ok(())
    }

    async fn list(&self, offset: u64, limit: u64) -> Result<(Vec<user::Model>, u64), DbErr> {
        use sea_orm::{EntityTrait, PaginatorTrait, QueryOrder};

        let paginator = User::find()
            .order_by_desc(user::Column::CreatedAt)
            .paginate(&self.db, limit);

        let total = paginator.num_items().await?;
        let users = paginator.fetch_page((offset / limit) as usize).await?;

        Ok((users, total))
    }
}
```

## src/domain/services/user.rs

```rust
use sea_orm::Set;

use crate::{
    domain::repositories::user::UserRepository,
    entities::user,
    utils::{auth, errors::AppError},
};

pub struct UserService<R: UserRepository> {
    repo: R,
    jwt_manager: auth::JWTManager,
}

impl<R: UserRepository> UserService<R> {
    pub fn new(repo: R, jwt_secret: String, jwt_expiration: i64) -> Self {
        Self {
            repo,
            jwt_manager: auth::JWTManager::new(jwt_secret, jwt_expiration),
        }
    }

    pub async fn register(
        &self,
        email: String,
        password: String,
        name: String,
    ) -> Result<(user::Model, String), AppError> {
        // Check if user exists
        if self.repo.find_by_email(&email).await?.is_some() {
            return Err(AppError::BadRequest("Email already in use".to_string()));
        }

        // Hash password
        let hashed_password = auth::hash_password(&password)?;

        // Create user
        let new_user = user::ActiveModel {
            email: Set(email.clone()),
            password: Set(hashed_password),
            name: Set(name),
            role: Set("user".to_string()),
            is_active: Set(true),
            ..Default::default()
        };

        let user = self.repo.create(new_user).await?;

        // Generate JWT
        let token = self
            .jwt_manager
            .generate(user.id, user.email.clone(), user.role.clone())?;

        Ok((user, token))
    }

    pub async fn login(
        &self,
        email: String,
        password: String,
    ) -> Result<(user::Model, String), AppError> {
        let user = self
            .repo
            .find_by_email(&email)
            .await?
            .ok_or_else(|| AppError::Unauthorized("Invalid credentials".to_string()))?;

        auth::verify_password(&user.password, &password)
            .map_err(|_| AppError::Unauthorized("Invalid credentials".to_string()))?;

        if !user.is_active {
            return Err(AppError::Forbidden("Account is deactivated".to_string()));
        }

        let token = self
            .jwt_manager
            .generate(user.id, user.email.clone(), user.role.clone())?;

        Ok((user, token))
    }
}
```

- **Zero-cost abstractions**: Iterators, closures compile to same code as manual loops
- **Error handling with Result<T, E>**: Use `?` operator for clean error propagation
- **Pattern matching**: Exhaustive matching ensures all cases handled
- **Trait system**: Enables generic programming without runtime overhead
- **Async/await**: Tokio runtime provides work-stealing scheduler for high concurrency
- **No GC pauses**: Predictable performance for latency-sensitive applications
