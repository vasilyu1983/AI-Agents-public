## Decision Trees (Quick Use)

```text
### AI Citation Rate Low
Check crawler access → Verify robots.txt allows GPTBot, ClaudeBot, PerplexityBot
Content too complex → Simplify structure, add clear headers, use direct answers
No structured data → Add FAQ, HowTo, Article schema markup
Missing entity signals → Strengthen topic authority, add entity-rich content

### High Impressions, Low Citations
Content lacks authority → Add data, research, expert quotes, original insights
Answer not direct → Rewrite with immediate value in first paragraph
Schema incomplete → Validate and enhance structured data
Factual accuracy issues → Fact-check, add sources, update stale data

### Platform-Specific Ranking Issues
ChatGPT not citing → Focus on LinkedIn, G2, Gartner-style content
Perplexity ignoring → Add video content, enhance PeerSpot-style reviews
Gemini missing → Optimize for Medium, Reddit, community platforms
Google AIO absent → Strengthen EEAT signals, add author bios

### Technical Issues
Crawlers blocked → Check robots.txt, security settings, JavaScript dependencies
JavaScript-heavy site → Add server-side rendering or static fallbacks
Dynamic content → Ensure critical content loads without JS
Slow response times → Optimize server performance for crawler requests

### Content Freshness
Stale citations → Update with 2024-2025 data, add recent examples
Old publication dates → Refresh content, update timestamps
Outdated facts → Verify against current sources, correct inaccuracies
```

For detailed decision trees, see individual platform strategy files in `resources/`.

### Key Differentiators

**AI Search vs Traditional SEO:**
- Traditional SEO: Optimize for Google crawlers, keywords, backlinks
- AI Search: Optimize for LLM retrieval, citations, answer quality

**Citation vs Ranking:**
- Citations (AI): Your content appears in LLM responses (6-10 sources per query)
- Rankings (Traditional): Your page appears in search result positions (#1-10)

**Content Focus:**
- Traditional: Long-form keyword optimization, link building
- AI Search: Direct answers, entity-rich, fact-checked, data-backed
